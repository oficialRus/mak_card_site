# Garmonia-MAK — Сайт метафорических ассоциативных карт

Премиальный ecommerce-сайт для бренда метафорических ассоциативных карт «Garmonia-MAK».

## Технологический стек

- **React 18 + TypeScript** — основа приложения
- **Vite 7** — сборка
- **Tailwind CSS 3** — стилизация
- **Framer Motion** — анимации (fade-in, slide-up, звёзды, парящие элементы)
- **Lucide React** — иконки (если нужны)

## Дизайн-концепция

**Двойная атмосфера:**
1. Тёплый золотой закат — персиковые, золотые, бежевые тона
2. Глубокий космический индиго — тёмно-синий, фиолетовый, звёздный фон

**Цветовая палитра:**
- Основной фон: `#050818` (глубокий космический)
- Акцентный индиго: `#080c2a`, `#0d1340`
- Золото: `#f5c842` (основной акцент), `#e8a937`
- Текст: `#f0e8d8`, `rgba(232,224,208,0.7)` для субтайтлов

**Типографика:**
- Заголовки: Cormorant Garamond (serif)
- Текст: Inter (sans-serif)

## Структура компонентов

```
src/components/
├── HeroSection.tsx      — Hero с анимированными звёздами и плавающими картами
├── ProductsSection.tsx  — Сетка трёх продуктов с hover-эффектами
├── PhilosophySection.tsx — Split layout: золотая и космическая стороны
├── AISection.tsx        — Секция AI-психолога с анимированным фоном
├── BenefitsSection.tsx  — 4 преимущества с золотыми иконками
├── ReviewsSection.tsx   — Glassmorphism отзывы на тёмном фоне
├── AboutSection.tsx     — Портрет автора + биография + статистика
├── FAQSection.tsx       — Аккордеон с анимациями framer-motion
└── FooterSection.tsx    — Тёмный футер с золотой типографикой
```

## Сборка

```bash
npm install
npm run build
```

## Особенности

- Все секции с плавными анимациями входа (whileInView)
- Минималистичные звёздные частицы в Hero и AI секциях
- Glassmorphism карточки в секции Reviews
- Кастомные CSS-классы: `.btn-gold`, `.btn-outline-gold`, `.glass-card`, `.text-gold-gradient`
- Адаптивная вёрстка (mobile-first)
- Плавный скролл через CSS `scroll-behavior: smooth`
